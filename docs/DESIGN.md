# Design

## Visual Theme
Warm neutral investment workspace with calm editorial rhythm, premium spacing, and a private-wealth feel.

## Color System
- Canvas: soft warm neutral backgrounds
- Surface: off-white panels with subtle border contrast
- Text: graphite / near-black primary copy with muted secondary text
- Accent: muted bronze / warm amber for calls to action and highlights
- Status: restrained green, amber, red, blue, and purple for semantic states

## Typography
- Primary: Manrope for UI, labels, navigation, and body copy
- Mono: JetBrains Mono for values, codes, timestamps, and system labels
- Hierarchy should be strong, compact, and editorial rather than playful

## Layout
- Max width: 1440px
- 12-column grid where practical
- Spacious section rhythm with clear decision hierarchy
- Sidebar / rail patterns for supporting context and quick actions
- Avoid dense consumer-finance dashboards and trading-terminal clutter

## Components
- MetricCard: top-level KPIs and decision metrics
- HoldingCard: portfolio positions and bucket items
- BucketSection: grouped holdings by portfolio category
- ActionCard: next best action and review prompts
- QueueItem: approval and decision queue rows
- ResearchCard: watchlist, notes, and ideas

## Motion
- Minimal, calm transitions
- Soft hover elevation only
- No flashy gradients or excessive motion

## Voice
- Confident, plainspoken, and calm
- Editorial and decision-first
- Suitable for a single long-term investor managing personal capital
