Spatial Runtime Foundation

Runtime spatial context, provider, hook, and component integration.

Architecture

SpatialProvider (OSLayout)
└── SpatialContext
    ├── config: { variant, major, minor, baseline, opacity, columns, rhythm }
    ├── isDebugMode
    └── toggleDebug()
        │
        ├── useSpatial() — hook for any component
        ├── GridOverlay — background grid (shell mode)
        └── GridDebug — dev overlay (Ctrl+Shift+G)

Flow

1
URL changes

useLocation() detects route change

↓
2
SpatialProvider derives config

variant from path: /os/studio → architect, else → operation

↓
3
Context provided to all children

GridOverlay renders at z-index 0 · GridDebug listens for Ctrl+Shift+G

↓
4
Components consume via useSpatial()

WorkspaceShell, WorkspaceSection, WorkspaceCard, MetricStrip, StatusRail
