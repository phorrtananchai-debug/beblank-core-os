Column System

12-column CSS Grid system. Sections declare columnSpan and columnStart. Common splits:

Hero:       8 / 4     (feature + rail)
Article:    7 / 5     (content + sidebar)
Overview:   3 / 6 / 3 (left rail + content + right rail)
Gallery:    4 / 4 / 4 (equal thirds)
Full:       12        (single column)

Baseline Grid

8px baseline grid. Toggleable via GridDebug (Ctrl+Shift+G). Layers:

  • Major grid (64px)
  • Minor grid (8px)
  • Column guides (12 columns)
  • Baseline (8px horizontal lines)
  • Margin guides (48px inline margins)

Layout Inspector

Press Ctrl+Shift+L to toggle. Shows:

  • Current variant, major/minor, baseline
  • Column count, opacity, rhythm values
  • Overlay layers (major, minor, columns, baseline, margins)
  • Each layer can be toggled individually
